This is a project about analyzing data and creating applications from given data.
RAG is used to analyze the data and create applications from given data using LLMs.
The data is stored in a database and the applications are created using the data.
The applications are created using the data and the applications are stored in a database.

The project is built using the following technologies:
- Python
- LLMs
- RAG
- Database
- streamlit

we are creating a simple stramlit based local json file for database and application.
